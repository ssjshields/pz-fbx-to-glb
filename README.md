# pz-fbx-to-glb
A portable CLI tool for converting FBX to GLB for Project Zomboid, with correct scale, UV safety, and ASSIMP compatibility.

1. extract anywhere, place *.fbx in "input" folder
2. run "convert.bat" and wait, *.glb will be generated in the "output" folder

# Requirements
Blender 4.5 - 5.x
